# 简介

OPEN-C3支持通过[内建插件](/内建插件/README.md)对各CDN厂商的CDN进行刷新。

支持的CDN厂商：
* > aliyun: 阿里云
* > aws: AWS
* > chinacache: 蓝汛
* > dnion: 帝联
* > exclouds: 逸云
* > huawei: 华为
* > isurecloud: 云端智度
* > jscdn: 金山云
* > qcloud: 腾讯
* > qingcdn: 白山云
* > qiniu: 七牛
* > ucloud: ucloud

# 使用

* > 学习OPEN-C3内建插件[cdnrefresh](/cdnrefresh/README.md)的用法。
