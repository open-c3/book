# 简介

OPEN-C3是一个开源的CI/CD/CO系统，已经在多个企业中使用。

欢迎使用，欢迎沟通，欢迎提交代码。

为了方便问题和需求的沉淀，请把问题提交到[issue](https://github.com/open-c3/open-c3/issues)中。
需要讨论的部分请到[聊天室](https://webchat.freenode.net/?channels=#open-c3)中进行。
系统会自动存档，方便后面的人查看。
